# Introdução ao Project Skills

Se tem uma das coisas que é muito importante que você aprenda, mas aprenda com profundidade, é trabalhar com **Skills** na hora de desenvolver aplicações com IA. E o porquê de eu estar dizendo isso? Porque os Skills se tornaram um padrão de mercado e uma ferramenta extremamente poderosa para dar superpoderes aos seus agentes de IA conseguirem codificar e fazer aquilo que você precisa, com uma especialidade específica.

E o mais interessante de tudo isso é que isso não vai encher, por padrão, a sua **janela de contexto**. Então, durante este capítulo, eu quero trazer para vocês uma visão geral do que são Skills. Depois disso, o que nós vamos fazer? Nós vamos olhar repositórios e projetos de Skills muito interessantes, que você pode explorar.

Também vou mostrar um Skill que eu criei aqui para que possamos ver, na prática, isso sendo utilizado. E também vamos criar juntos um Skill para que possamos, vamos dizer assim, sentir um pouco na pele quais são os principais desafios na hora de trabalhar com isso. Então, vamos nessa, porque tem bastante coisa para a gente ver.
