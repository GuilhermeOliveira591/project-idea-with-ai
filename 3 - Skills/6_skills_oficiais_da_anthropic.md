# Skills oficiais da Anthropic

Vamos sair um pouco do Excalidraw e ir mais para a prática agora, para explorarmos. Vamos navegar um pouco na internet, e eu vou dar algumas dicas e passar alguns links para vocês também, para que você tenha referências para acessar.

Então, o seguinte: eu tenho aqui a primeira coisa que queria mostrar. Tenho um repositório — deixa eu aumentar a tela — que é um **repositório oficial de Skills da Anthropic**. Então, aqui está o endereço: `anthropic-skills`, da organização Anthropic, barra skills. E aqui existem algumas Skills oficiais que eles utilizam e que você também pode utilizar.

Eu tenho aqui uma skill para trabalhar com `DOCX`, ou seja, arquivos do Word, porque muitas vezes preciso fazer parse ou criar documentos nesse formato. Às vezes, também preciso saber lidar com `PDF`. Então, existe uma skill específica para trabalhar com PDF.

Você já tentou trabalhar com PDF usando IA e ela ficou tentando criar scripts em Python de várias formas, carregar bibliotecas diferentes ou tentar ler o arquivo de maneiras diversas? Provavelmente, essa skill consegue ajudar. A mesma coisa acontece com **PowerPoint**.

Agora, o mais interessante de tudo isso é perceber que **as skills não são algo feito apenas para desenvolvimento de software**. Elas são voltadas para o uso de inteligência artificial de forma geral. Se você abrir o seu Claude, não sei se você é assinante do Claude, não estou falando do Claude Code, estou falando do aplicativo do Claude, ou se você utiliza outras ferramentas — acredito que o ChatGPT também tenha — você pode carregar skills nessas ferramentas.

E o que isso significa? Significa que você pode ter skills específicas, inclusive essas aqui, para que, ao utilizar uma ferramenta de IA que não seja um editor de código, você consiga fazer com que ela trabalhe com essas funcionalidades. Então, existem skills até para *branding*, criação de marca, uso de canvas para desenho e também para trabalhar com **Excel**.

E tem, por exemplo, o **MCP Builder**. Quem já sofreu para criar um servidor MCP sabe que dá bastante trabalho, porque existe muita documentação distribuída e diversos tipos de MCP. Existe *HTTP Streamable*, existe *STDIO*, existem diferentes formatos e organizações. Então, aqui temos um MCP Builder como skill também.

Então, o que eu quero trazer para vocês é que isso vai muito além do mundo do desenvolvimento. E uma das coisas que vamos experimentar hoje é que a Anthropic criou um **Skill Creator**. E o que é esse Skill Creator? É uma forma de criar uma skill através de outra skill.

Isso é interessante porque o Skill Creator pode fornecer Guidelines, fazer perguntas e reunir todas as informações necessárias para, no final, gerar os arquivos da skill que você precisa. Vamos utilizar essa ferramenta ainda hoje.

Essas são algumas das skills da Anthropic. Eu vou mostrar outros sites em seguida e depois vamos começar a abrir as skills, analisar os arquivos, explorar os conteúdos e discutir alguns aspectos. Eu clonei diversos repositórios e preparei alguns exemplos para a gente trabalhar. Então, vamos nessa.
